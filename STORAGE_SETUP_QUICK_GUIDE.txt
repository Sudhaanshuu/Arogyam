🎯 STEP-BY-STEP INSTRUCTIONS

1. Open Supabase Dashboard → https://app.supabase.com
2. Select your Arogyam project
3. Click "Storage" in left sidebar
4. Click "New bucket" button (top right)
5. For EACH bucket:
   
   ┌─────────────────────────────────────┐
   │ Create a new bucket                 │
   ├─────────────────────────────────────┤
   │ Name: [bucket-name]                 │
   │                                     │
   │ ☑/☐ Public bucket                   │
   │     (check for public, uncheck for  │
   │      private)                       │
   │                                     │
   │ [Create bucket]                     │
   └─────────────────────────────────────┘

6. Repeat for all 6 buckets

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❓ WHY THIS ERROR HAPPENED

Error: "must be owner of table buckets"

→ Storage buckets can ONLY be created through the Supabase Dashboard UI
→ The SQL INSERT commands don't work due to permission restrictions
→ This is by design for security reasons
