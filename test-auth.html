🔐 Arogyam Authentication System Test

Use this page to test if the authentication system is working correctly.

Step 1: Check Auth Debug Tools

Step 2: Check Current Session

Step 3: Check Zustand Store

Step 4: Clear Auth Cache (if issues)

Step 5: Force Sign Out (if issues)

Instructions

  1. Open your app in another tab (http://localhost:5173 or your dev server)
  2. Keep this tab open with dev tools console visible (F12)
  3. Click "Check Debug Tools" to verify utilities are loaded
  4. Try logging in on your app tab
  5. Come back here and click "Check Session" to see if you're logged in
  6. Click "Check Store State" to see if user data is in the store
  7. If issues: Use "Clear Cache" and try again

Expected Results

✅ After login, "Check Session" should show:
- Session exists: true
- User ID: [your user id]
- User email: [your email]

✅ "Check Store State" should show:
- user: { id: "...", email: "..." }
- profile: { id: "...", full_name: "..." }
- loading: false
- initialized: true

❌ If session exists but store.user is null:
→ Store not updating properly, check console logs
